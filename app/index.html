How to deploy (60 seconds)
  1. Unzip the downloaded file
  2. Create a new GitHub repo named <yourusername>.github.io
  3. Push the files: git init && git remote add origin <repo-url> && git add . && git push -u origin main
  4. Go to repo Settings → Pages → Source: main → Save
  5. Your portfolio is live at https://<yourusername>.github.io
