Deploy steps
  1. Unzip the downloaded file
  2. Create a GitHub repo named <yourusername>.github.io
  3. Push: git init && git remote add origin <repo-url> && git add . && git commit -m "Initial portfolio" && git push -u origin main
  4. GitHub repo → Settings → Pages → Source: main → Save
  5. Live at https://<yourusername>.github.io

After that, just edit data.js and push — CI/CD handles the rest.
