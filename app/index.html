Step 3 — AI-powered parsing optional

Leave off to use the built-in rule-based parser (no API key needed). Enable to let an LLM extract and enrich your content — much more accurate, especially for complex resume layouts.

Use AI parsing
No API key? Use Ollama — free, runs on your machine, no account needed.
Install Ollama (takes ~2 min)
  1. Mac/Linux — run in Terminal:
    curl -fsSL https://ollama.com/install.sh | sh
  2. Windows — download installer from ollama.com/download
  3. Start Ollama, then pull a model:
    ollama pull llama3.2  (~2 GB, recommended)
    ollama pull llama3  (~4 GB, more capable)
  4. Come back here, pick Ollama — local as provider, leave API key blank, and generate.

Your API key is sent directly from your browser to the backend running locally. It is never stored or logged.
